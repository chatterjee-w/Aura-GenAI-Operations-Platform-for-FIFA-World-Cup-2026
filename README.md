# Aura - FIFA Nexus GenAI Operations Platform

Aura is a multimodal GenAI copilot for FIFA World Cup 2026 — one system powering wayfinding, crowd management, accessibility, transportation, and multilingual guest services simultaneously.

## Development & Testing

This project uses modern JavaScript tooling to ensure code quality and prevent regressions.

### Prerequisites
- Node.js (v18+)

### Setup
```bash
# Install dependencies
npm install
```

### Running Tests
We use Vitest and JSDOM to test the vanilla JS modules and DOM interactions.
```bash
# Run the unit tests
npm test
```

### Linting
We use ESLint to enforce code quality and standard conventions.
```bash
# Run the linter
npm run lint
```
